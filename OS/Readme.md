## CSAPP:

CMU 镇系神课, 课程内容覆盖了编码、体系结构、操作系统、编译链接、网络、并发等. CSAPP 可以看作是计算机科学的总览.
全书有三个核心的视角:

1. 程序视角: 编译链接, 地址重定位
2. 硬件CPU视角: 指令的执行, 存储结构
3. 操作系统视角: 程序加载成为进程, 进程管理

参考:
https://csdiy.wiki/%E4%BD%93%E7%B3%BB%E7%BB%93%E6%9E%84/CSAPP/
